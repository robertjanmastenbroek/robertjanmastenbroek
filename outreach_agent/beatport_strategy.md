## PART 1 — BEATPORT

### Why Beatport Matters
- DJs buy and play tracks they find on Beatport
- Beatport Top 100 placements drive organic Spotify growth
- Press (RA, DJ Mag, 5Mag) use Beatport charts for coverage leads
- Label distribution to Beatport requires a release on an approved label

### How to Get on Beatport

**Option A — Via an Existing Label (Fastest)**
RJM is currently on Spotify via self-distribution (DistroKid/TuneCore). Beatport requires label distribution — most self-distributed artists are NOT automatically on Beatport.

Steps:
1. Check if current distributor (DistroKid/TuneCore) offers Beatport distribution — DistroKid does via "Beatport & Traxsource" add-on ($7.95/yr extra per release)
2. If already enabled: re-submit current catalogue to Beatport via distributor dashboard
3. If not enabled: upgrade plan and submit 3–5 strongest tracks first (Halleluyah 140 BPM, Jericho 140 BPM, Living Water 124 BPM)

**Option B — Sign to a Small Label**
Several psytrance/melodic techno labels are Beatport-approved and actively seeking demos:
- Yoyo Records (psytrance)
- Digital Om Productions (goa/psytrance)  
- Dacru Records (psytrance, Belgium)
- Altar Records (psytrance)
- Yellow Sunshine Explosion (goa)

Label deal = Beatport presence + free PR + potential advance

**Recommended first step:** Enable Beatport via DistroKid add-on for existing catalogue.

### Beatport Genres to Target
- Psytrance (Halleluyah, Jericho) → aim for Psytrance Top 100
- Melodic House & Techno (Living Water) → aim for MH&T Top 100
- Progressive House (Renamed) → Progressive House chart

### Promotion on Beatport
- Run Beatport Featured placement ($50–200/slot) once charting

## PART 2 — SOUNDCLOUD

### Why SoundCloud
- Curators, DJs, and labels discover new artists here
- SoundCloud reposts reach niche audiences instantly
- Free tracks/extended mixes allowed (Spotify doesn't allow free downloads)
- SoundCloud Go+ pays per stream as secondary revenue

### SoundCloud Upload Checklist

**Account Setup:**
- [ ] Create verified SoundCloud Pro account ($8/month or $6/month annual)
- [ ] Set artist bio with Tenerife / Bible-inspired story (2–3 lines)
- [ ] Add all social links (IG, Spotify, website)
- [ ] Set profile image (same as all other platforms — brand consistency)
- [ ] Add header/banner image

**Track Upload Standards:**
- [ ] WAV file (minimum 24-bit/44.1kHz — SoundCloud masters at 128kbps but stores original)
- [ ] Proper metadata: Title, Artist, Genre, BPM, Key, Label
- [ ] Description: 3–4 lines about the track's story/inspiration (biblical angle — subtle salt)
- [ ] Tags: psytrance, tribal techno, melodic techno, Robert-Jan Mastenbroek, Tenerife, [BPM]bpm, [key]
- [ ] Set Release Date to match Spotify release
- [ ] Private preview link for demo sends (to labels/curators before public release)

**Content Strategy on SoundCloud:**
1. Upload each new weekly release on the same day it hits Spotify
2. Upload one monthly extended mix (60–90 min) — psytrance/tribal set
3. Upload one "studio session" recording per month (raw, unmastered — exclusive to SC)
4. Keep one permanent free download track to grow mailing list
