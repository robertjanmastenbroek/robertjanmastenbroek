# RJM Release Checklist — Every Single Release, No Exceptions
### Permanent operations checklist for Robert-Jan Mastenbroek releases

---

## 14+ DAYS BEFORE RELEASE

### Spotify for Artists — Editorial Pitch

- [ ] Log into artists.spotify.com
- [ ] Confirm unreleased track is visible in the dashboard (distributor must have delivered it)
- [ ] Click **"Pitch a Song"** on the track — this opens the editorial pitch form
- [ ] Complete ALL fields — incomplete pitches are deprioritised by the editorial team

**MOOD TAGS** (select up to 3 — use these for RJM's music):
- For 120–125 BPM tracks (Living Water, He Is The Light, You See It All): `Peaceful` + `Spiritual` + `Intense`
- For 130 BPM tracks (Renamed, Fire In Our Hands): `Energetic` + `Spiritual` + `Intense`
- For 140 BPM tracks (Halleluyah, Jericho): `Aggressive` + `Intense` + `Energetic`

**STYLE TAGS** (select up to 3):
- All tracks: `Hypnotic` + `Dark` + `Tribal`
- Lighter tracks: swap `Dark` for `Ethereal`

**INSTRUMENTS** (list all that appear in the track — be exhaustive):
- Standard set: `Synthesizer, Drum Machine, Bass, Percussion, Vocals (chanting), Kick Drum, Hi-Hats`
- Add if present: `Piano, Strings, Guitar, Field Recording, Spoken Word`

**LANGUAGE FIELD:**
- Set to `Hebrew` if Hebrew chanting is present (Renamed, Halleluyah, Jericho)
- Set to `No Lyrics / Instrumental` for pure instrumentals

**STORY FIELD** — use this exact text as the base, customise per track:

```
Dutch producer who rebuilt his life in Tenerife after losing a €6M startup. 
Robert-Jan Mastenbroek makes Nomadic Electronic — organic powerful house through 
tribal psytrance, oud and handpan over ancient Middle Eastern rhythms. Ancient 
truth carried by future sound. [TRACK NAME] is [brief track description: e.g. 
"a 140 BPM tribal psytrance exploration of the Hebrew name for rebirth" or 
"a 124 BPM organic house meditation on John 4"]. Pitched for: [target playlist names].
```

**GENRE FIELD:**
- Primary: `Electronic`
- Secondary: `Dance/Electronic`
- Do NOT set primary to just "Techno" — it narrows editorial reach significantly

**PLAYLISTS TO MENTION IN PITCH:**
- `Fresh Finds: Dance/Electronic`
- `Tribal Techno`
- `Sacred/Spiritual`
- `Any Word` (faith-based electronic playlist)
- `Electronic Rising`
- `Altar Sounds` (if applicable)

### DistroKid / Distributor Metadata

Complete these fields exactly before submitting to distributor:

- [ ] **Track Title:** Use spiritual/evocative language (see naming guide below)
- [ ] **Artist Name:** `Robert-Jan Mastenbroek` — no variations, no abbreviations
- [ ] **Primary Genre:** `Electronic`
- [ ] **Secondary Genre:** `Dance/Electronic`
- [ ] **Release Date:** Always a Friday
- [ ] **Language:** Hebrew (if chanting present) or English or Instrumental
- [ ] **BPM:** Tag correctly — check in DAW before submitting
- [ ] **ISRC Code:** Auto-generated by DistroKid — save in your release log
- [ ] **Lyrics:** Upload .txt lyrics file if any vocals/chanting present (increases playlist eligibility)
- [ ] **Explicit content:** Mark correctly (most RJM tracks: Not Explicit)
- [ ] **Pre-order / Pre-save:** Set up pre-save at least 7 days before release

**Track naming guide:**
- Good: "Jericho", "Living Water", "Renamed", "Halleluyah", "He Is The Light"
- Good pattern: [Biblical place / concept] OR [Biblical phrase] OR [Hebrew word]
- Bad: "Track 01", "Demo Final 3", "New Mix 2026", "WIP Psytrance"

**Genre tags to use in distributor system:**
`Nomadic Electronic, Organic-Tribal House, Tribal Psytrance, Ethnic Electronic, Sacred Electronic, Desert House, Middle Eastern Electronic, Handpan Electronic, Underground Electronic`

### Spotify Canvas — Specs and Content

- [ ] Create an **8-second looping vertical video** — specs: 1080×1920 px, MP4, max 8MB
- [ ] Upload via Spotify for Artists → select track → Canvas tab
- [ ] Canvas must loop seamlessly — test it before uploading

**What to show (by track energy):**

| Track BPM | Canvas Content Ideas |
|-----------|---------------------|
| 120–124 BPM | Slow ocean waves, Atlantic sunrise/sunset, candle flame, flowing water |
| 130 BPM | Time-lapse Tenerife clouds, desert wind across sand, slow fire |
| 140 BPM | Sacred geometry animation (geometric loop), lightning in slow motion, dark forest |

**Tools:** CapCut (free, mobile), Canva Pro (loop video), Adobe Premiere, DaVinci Resolve
**Rule:** No faces, no text overlays, no brand logos — pure atmosphere only

- [ ] Test the pre-save link from a personal Spotify account

## POST-RELEASE (Days 2–7)

### Day 2 — Follow-Up Content

- [ ] Post a "reactions" Story: screenshot real comments/DMs (with permission) + "This is why I make music"
- [ ] Post a follow-up Story with the stat: "X people saved [TRACK NAME] in the first 24 hours" — creates social proof

### Day 3 — Follow-Up Feed Post

Use Pillar 4 / Caption 4C template:
- Screenshot Spotify stream count (or a listener comment)
- Reframe the track from a different angle (if Day 1 was about the biblical reference, Day 3 post focuses on the sonic journey)
- CTA: "Still haven't heard it? Link in bio. It gets better on every listen."

### Days 4–7 — Playlist Pitching Based on Early Streams

- [ ] Check Spotify for Artists → Song stats → Streams on day 1, 2, 3
- [ ] If 500+ streams in first 3 days: screenshot the chart and pitch to curators with: "Released Friday, 500+ streams organically in 72 hours — [Track Name] is landing."
- [ ] If 200–500 streams: normal pitch, focus on quality and genre fit
- [ ] If under 200 streams: do NOT mention stream numbers — focus entirely on genre fit and story
- [ ] Submit to curators via outreach agent pipeline (see agent.py)
- [ ] Submit to SubmitHub (see `submithub_targets.md`)
- [ ] Submit to Groover for blog and playlist coverage
