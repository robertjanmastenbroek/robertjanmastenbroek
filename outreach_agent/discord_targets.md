## Strategy Notes

Discord psytrance/techno communities don't operate curator-style submission inboxes. The play is:
**Join → be active → drop tracks in self-promo channels → network with playlist runners/YouTube curators inside.**

This is a human-loop task — not automatable via the Gmail outreach agent. Recommend a manual 30-minute weekly schedule: drop one track in each active server's self-promo channel with a short, visual, non-spammy message.

Re-verify invite links quarterly via DISBOARD / Discodus / Top.gg — they churn.

## Drop Script Template (self-promo channels)

Use this format — visual, short, never spammy:

> **New track — "[TRACK NAME]" — 140 BPM Psytrance / [genre]**
>
> Recorded in Tenerife. Mastered live in the studio. [ONE visual detail — "the synth sat in the sun for 6 hours before the session"].
>
> Spotify: [link] | SoundCloud: [link]
>
> Happy to give feedback on yours too — drop it below.

Rules:
- Never use "blessed" / "anointed" / "spiritual journey"
- One track per server per week, max
- Always offer reciprocal feedback — this gets replies
- Visual + specific language only (BPM, location, specific detail)
