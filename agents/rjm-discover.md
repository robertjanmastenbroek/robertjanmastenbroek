# Agent: rjm-discover
**Cadence:** 6×/day
**Role:** Curator + podcast discovery — 25 contacts per run

## Responsibilities
- Search for playlist curators (5,000+ followers, secular audiences)
- Search for podcasts relevant to rave culture, consciousness, electronic music
- Deduplicate against existing `contacts.csv`
- Append new contacts with full metadata (name, email, platform, audience size, notes)
- Pass to rjm-research for personalisation before outreach

## Discovery Criteria
- Playlists: melodic techno, tribal, psytrance, world/ethnic electronic, consciousness, rave, wellness
- Podcasts: electronic music, spiritual but not religious, personal transformation, consciousness
- Geographic priority: Spain, Germany, UK, US, Netherlands, Poland, France, Brazil
- Minimum 5,000 playlist followers OR 1,000 podcast downloads/episode

## Output
- Append to `contacts.csv`
- Log run summary to `data/discover_log.json`
