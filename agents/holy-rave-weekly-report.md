# Agent: holy-rave-weekly-report
**Cadence:** Weekly
**Role:** Analytics pulse — Spotify KPI primary, Buffer secondary

## Report Structure
1. **Spotify KPIs** (primary)
   - Monthly listeners (vs. 1M target)
   - Stream count by track
   - Playlist adds this week
   - Saves and follows
   - Top markets (Spain, US, Germany, UK, Poland, NL, France, Brazil)

2. **Buffer / Social** (secondary)
   - Views per clip (TikTok, IG, YouTube)
   - Best-performing hook
   - Follower delta

3. **Outreach Pipeline**
   - Emails sent this week
   - Reply rate
   - Curator placements secured
   - Podcast bookings confirmed

4. **Action Items**
   - What to double down on
   - What to cut
   - Next week's priority

## Output
- Save to `RJM_Daily_Report.md` (overwrite)
- North Star progress: current listeners / 1,000,000
